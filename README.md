# 🎬 Movie Industry Analysis Dashboard

 📌 Overview
This project explores trends across 7,000+ movies to understand relationships between budget, genre, ratings, and profitability, with insights presented through an interactive dashboard.

 🛠️ Tools & Technologies
- Python (Pandas, Matplotlib)
- Power BI
- Excel

 📂 Dataset
- From Kaggle dataset collection
- 7,000+ movie records
- Includes genre, budget, revenue, ratings, and release year and country 

 🔍 Key Analysis
- Cleaned and transformed raw data using Pandas  
- Performed Exploratory Data Analysis to identify trends and patterns  
- Analysed correlations between budget, ratings, genre, and profitability via visualisation and Matplotlib
- Investigated industry trends across time and global markets  

 📈 Key Insights
- Higher budgets do not always guarantee higher profitability which can not also ensure higher Return on Investment (ROI)
- Certain genres consistently outperform others in ROI and number 
- High profitability does not guarantee the industry and directors preference over producing them 
- Movie production trends hence budget vary significantly across years  
- Profit and ROI generally have positive correlation throughout the 40 year period 
- High rating does not guarantee high profit but high profit generally comes with high ratings
- Children movies and Animation account for a significant part of the whole dataset profit
- Despite making positive industry profit, just the few top countries are making huge profit out of this industry including familiar names like United States, South Korean or Hong Kong, the remaining smaller countries have been incurring negative profit
